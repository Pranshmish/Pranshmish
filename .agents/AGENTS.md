# GitHub Profile README Rules

## Project Overview
This repository is a GitHub profile README repository (Pranshmish/Pranshmish). The `README.md` file in this repo is displayed on the GitHub profile page.

## Design Rules
1. **Premium Aesthetic** — All SVG assets must look like a $500 GitHub profile header. Extremely polished, minimal, elegant, modern, pixel-perfect.
2. **No JavaScript in SVGs** — GitHub strips JS from SVGs. Use only SMIL animations (`<animate>`, `<animateTransform>`, `<animateMotion>`).
3. **Glassmorphism** — Use background blur, soft reflections, rounded corners, thin glowing borders.
4. **Dark + Light Mode** — Always provide both `dark.svg` and `light.svg` variants using `prefers-color-scheme` media query in README.
5. **Color Palettes:**
   - Dark: Background `#030712`, Panels `#0F172A`, Accent gradient `#7C3AED → #22D3EE → #10B981`
   - Light: Background `#FFFFFF`, Panels `#F8FAFC`, Accent gradient `#2563EB → #06B6D4 → #10B981`
6. **Animations** — Everything must animate: typing, scanlines, gradient shifts, floating particles, glow pulse, sequential reveal.
7. **SVG Only** — Use `linearGradient`, `radialGradient`, `filter`, `mask`, `clipPath`, `animate`, `animateTransform`.
8. **No placeholders** — All content must be real and accurate.
